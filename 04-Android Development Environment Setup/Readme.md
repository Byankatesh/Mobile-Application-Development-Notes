📌 Introduction

Setting up the Android development environment is the first step in building Android apps. It includes installing tools, setting up the IDE, and preparing devices for testing.

🛠️ Install Android Studio

Android Studio is the official IDE provided by Google.

🔽 Download & Install

Go to Android Studio official website

Download installer

Run setup

Install:

Android SDK

SDK Tools

Emulator

🖼️ Android Studio Setup Screen
⚙️ Configure Android Studio

After installation:

Open Android Studio

Select Standard Setup

Choose theme (Light/Dark)

Download required SDK packages

📱 Android Virtual Device (AVD)

An AVD (Android Virtual Device) is an emulator that acts like a real mobile device.

🧠 Why use AVD?

No need for physical device

Test different Android versions

Easy debugging

🛠️ Create AVD (Step-by-Step)

Open Android Studio

Go to Device Manager

Click Create Device

Select device (Pixel, etc.)

Choose Android version

Click Finish

🖼️ AVD Creation Process
▶️ Run App on Emulator

Select AVD

Click Run ▶️

App will launch on emulator

📲 Run App on Real Device
Steps:

Enable Developer Options

Turn ON USB Debugging

Connect device via USB

Allow permission

Click Run ▶️

⭐ Key Features

Fast development with emulator

Setting up Android Studio and AVD helps developers build and test apps efficiently. It is the foundation of Android app development.
